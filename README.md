# 转换：共性，形式，及机制

**人类文明如何组织世界、重写世界、并在重写中获取力量的方法论**

> *从一种形式到另一种形式的转换，不是游戏，是杠杆。*
>
> *Transformation from one form to another is not a game—it is a lever.*

---

## 关于本书

同一个内容，常常可以存在于不同形式之中。不同形式各有长短；而一旦找到稳定的转换机制，人类就能把原本困在一种形式中的对象，送入另一种更适于处理的空间，获得新的能力、新的速度、新的可见性。与此同时，每一次转换都带着代价：损耗、压缩、删减、遮蔽、误读，甚至傲慢。

本书以 **共性 → 形式 → 机制 → 代价** 为主线，穿越十个跨学科案例，揭示人类文明一条隐秘的主线：不断发现共性、发明形式、建立机制，并承受其代价的历史。

> 人类文明并不只是不断发现新对象的历史，更是不断为对象找到新写法的历史。
